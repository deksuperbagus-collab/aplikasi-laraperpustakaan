# aplikasi-laraperpustakaan
Aplikasi Sistem Informasi Perpustakaan berbasis Laravel yang digunakan untuk mengelola data buku, member, peminjaman dan pengembalian buku, serta laporan buku yang paling banyak dipinjam. Proyek ini dibuat sebagai project tengah perkuliahan dengan tampilan sederhana menggunakan Bootstrap dan tanpa autentikasi pengguna.
